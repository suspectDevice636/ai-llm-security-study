## 🧪 Labs & Practice

### Four Attack Vectors Covered

1. **Prompt Injection & Jailbreaking** — Overriding LLM instructions with malicious prompts
2. **Data Extraction & Privacy Attacks** — Extracting training data, system prompts, and sensitive information
3. **Model Poisoning & Evasion** — Bypassing safety controls and detection systems
4. **Advanced Attacks** — Chained attacks, edge cases, and emerging techniques

### What's Included

✅ **Hands-On Labs** — 4 complete modules with executable examples
✅ **Practice Questions** — 30+ questions with detailed answers
✅ **Code Examples** — Python scripts for prompt injection, detection, and analysis
✅ **Real-World Scenarios** — Case studies and practical applications
✅ **Testing Checklists** — Structured methodologies for security assessment
